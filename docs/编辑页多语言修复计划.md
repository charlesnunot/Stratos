# 编辑页多语言修复计划

**问题**：访问 `/en/seller/products/[id]/edit` 时，页面出现中文，且商品数据全部显示为中文。

**目标**：在 `/en/` 路由下，界面和商品数据应优先显示英文；在 `/zh/` 下显示中文。

---

本计划已拆分为 3 个文档：

1. **[01-问题分析](./编辑页多语言修复-01-问题分析.md)** — 问题来源、商品翻译字段说明
2. **[02-UI与翻译](./编辑页多语言修复-02-UI与翻译.md)** — UI 硬编码替换、翻译 key 补充、实施步骤
3. **[03-商品数据与提交逻辑](./编辑页多语言修复-03-商品数据与提交逻辑.md)** — 按 locale 显示商品数据、handleSubmit 修改、验证步骤
