## 二、国家选择器组件

### 2.1 新建组件

**路径**：`src/components/ecommerce/SalesCountriesSelector.tsx`

**职责**：
- 接收 `value: string[]`、`onChange: (codes: string[]) => void`、`disabled?: boolean`
- 多选 UI：可用 Checkbox 列表或带搜索的多选下拉（如项目有 shadcn Command 可复用）
- 根据 `useLocale()` 显示 `nameZh` 或 `nameEn`
- 空选择表示「未限制」

**Props 接口**：

```ts
interface SalesCountriesSelectorProps {
  value: string[]
  onChange: (codes: string[]) => void
  disabled?: boolean
  placeholder?: string
}
```

**可选增强**：
- 「全选」「清空」快捷按钮
- 按地区分组（亚洲、欧洲、美洲等）提升可读性
