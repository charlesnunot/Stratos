## 阶段 2：补充缺失翻译 key

在 `src/messages/zh.json` 和 `src/messages/en.json` 的 `seller` 命名空间下新增：

```json
{
  "seller": {
    "faqQuestion": "问题" / "Question",
    "faqAnswer": "答案" / "Answer",
    "productImages": "商品图片" / "Product Images",
    "existingImages": "现有图片" / "Existing Images",
    "externalImageUrls": "图片外链" / "External Image URLs"
  }
}
```

在 `common` 命名空间（若尚未存在）新增：

```json
{
  "common": {
    "sold": "已售罄" / "Sold Out"
  }
}
```

## 实施步骤

1. 在 `src/messages/zh.json` 和 `src/messages/en.json` 的 `seller` 中添加：`faqQuestion`、`faqAnswer`、`productImages`、`existingImages`、`externalImageUrls`
2. 在 `common` 中检查并添加 `sold`（中英）
3. 将 `edit/page.tsx` 中所有硬编码中文替换为 `t('key')` 或 `tCommon('key')`
4. 移除所有 `|| '中文fallback'` 形式的 fallback
