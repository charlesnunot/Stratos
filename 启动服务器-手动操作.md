# 手动启动开发服务器

## ⚠️ 问题

自动启动遇到权限错误，建议手动操作。

## 🚀 手动启动步骤

### 方法1：在PowerShell中直接运行（推荐）

1. **打开新的PowerShell窗口**
   - 按 `Win + X`
   - 选择 "Windows PowerShell" 或 "终端"

2. **运行以下命令**：
   ```powershell
   cd C:\Users\admin\Desktop\Stratos
   npm run dev
   ```

3. **等待启动**
   - 看到 "Ready" 消息表示成功
   - 访问 http://localhost:3000

### 方法2：如果遇到权限问题

1. **以管理员身份运行PowerShell**
   - 右键点击 PowerShell
   - 选择 "以管理员身份运行"

2. **然后运行**：
   ```powershell
   cd C:\Users\admin\Desktop\Stratos
   npm run dev
   ```

### 方法3：如果Node.js版本问题

您当前使用的是 Node.js v22.19.0，可能太新了。

**推荐使用 Node.js 20 LTS**：

1. 访问：https://nodejs.org/
2. 下载 Node.js 20.x LTS 版本
3. 安装后重启终端
4. 运行 `npm run dev`

## ✅ 成功标志

启动成功后，您会看到：

```
▲ Next.js 14.0.4
- Local:        http://localhost:3000
- Ready in 2.5s
```

## 🔍 如果仍然失败

请告诉我具体的错误信息，我会帮您解决！

常见错误：
- 端口被占用 → 使用 `$env:PORT=3001; npm run dev`
- 依赖缺失 → 运行 `npm install`
- 权限错误 → 以管理员身份运行
